This is a cipher app that allows you to encode your message in one of the following ways:
<ul>
<li> Caesar cipher </li>
<li> Substitution cipher </li>
<li> Transposition cipher </li>
<li> Playfair cipher </li>
<li> Vigenere cipher </li>
</ul>
